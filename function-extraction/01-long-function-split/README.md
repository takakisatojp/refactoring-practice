# 01: Long Function Split（長い関数の分割）

## 学習目標

- ✅ 長い関数を意味のある単位で分割
- ✅ 各関数の責務を明確に
- ✅ テストしやすいコード設計

## 問題点

1つの関数で複数の処理を行うと、テストが困難で保守しにくくなります：

```typescript
// 100行以上の巨大関数
function processUserRegistration(userData) {
  // 入力検証
  // データベース操作
  // メール送信
  // ログ記録
  // すべてが混在...
}
```

## ゴール

関数を適切に分割して、以下を達成してください：

1. ✅ 1つの関数 = 1つの責務
2. ✅ 各関数が10～50行程度のサイズ
3. ✅ テストしやすくなる

---

**難易度**: ⭐ Easy | **所要時間**: 15-20分
